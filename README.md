# Сервис жеребьёвки (round-robin)

SPA для жеребьёвки шахматных турниров по круговой системе. Без бэкенда: данные сохраняются в браузере.

## Локальный запуск (без Docker)

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`.

## Тесты

```bash
npm run test
```

## Запуск через Docker

Сборка и запуск контейнера:

```bash
docker build -t round-robin-draw-service .
docker run -p 8080:80 round-robin-draw-service
```

Или через Docker Compose:

```bash
docker compose up --build
```

Откройте `http://localhost:8080`.

## Развёртывание

Контейнер отдаёт статические файлы через `nginx`. Можно разместить на любом сервере/хостинге с поддержкой Docker:

1. Соберите образ или запустите `docker compose up --build`.
2. Пробросьте порт 80 контейнера наружу.
3. Убедитесь, что прокси/балансировщик пропускает запросы к `/` и статическим файлам.

## Примечания

- Результаты вводятся через модальное окно по клику на ячейку тура.
- BYE не даёт очков (у всех игроков по одному BYE при нечётном числе участников).
- Доп.показатель: коэффициент Бергера.
