# Сервис жеребьёвки турниров по круговой системе

Ссылка на сервис: https://nokvix.github.io/round-robin-tournament-draw/

## Функционал

- Создание турнира: название, дата, главный судья, список участников.
- Генерация круговой жеребьёвки по таблицам Бергера.
- Таблица результатов в формате «игрок × игрок» с вводом результатов.
- Подсчёт очков и коэффициента Бергера.
- Ручное редактирование мест (поддержка диапазонов типа `1-2`).
- Печатные формы:
  - таблица результатов;
  - пары туров.
- Экспорт/импорт в JSON, автосохранение в localStorage.

## Локальный запуск (без Docker)

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`.

## Запуск через Docker

Сборка и запуск контейнера:

```bash
docker build -t round-robin-draw-service .
docker run -p 8080:80 round-robin-draw-service
```

Откройте `http://localhost:8080`.

## Запуск через Docker Compose

```bash
docker compose up --build
```

Откройте `http://localhost:8080`.

## Развёртывание

Контейнер отдаёт статику через `nginx`. Для размещения на сервере:

1. Соберите образ или используйте `docker compose up --build`.
2. Пробросьте порт контейнера наружу (пример: `-p 8080:80`).
3. Убедитесь, что внешний порт открыт в firewall/SG.

## Примечание

- BYE не даёт очков и отображается в списке пар как «свободен».
- Печатную форму пар можно открыть в отдельной вкладке (кнопка «Открыть пары на туры»).
- Результаты вводятся через модальное окно по клику на ячейку тура.
- Дополнительный показатель: коэффициент Бергера.
